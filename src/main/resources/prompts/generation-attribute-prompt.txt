{
  "metadata": {
    "entity_type": "variable",
    "maintenance_agency": "BIRD",
    "id": "...",
    "code": "...",
    "title": "...",
    "domain_id": "...",

    "valid_to": "9999-12-31",
    "description": "..."

- title must be in English, in Title Case
- domain_id must be coherent with the description, also abbreviated following the same code pattern as "code" but applied to domain name
- valid_from must be today's date in YYYY-MM-DD format
- Respond STRICTLY in JSON, no markdown fences, no extra text before or after.
